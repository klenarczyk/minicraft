## Project Modules

<CardGrid stagger>
    <Card title="Core Engine" icon="laptop">
        The backbone of the project, handling the Game Loop, Windowing, and
        low-level resource management. Located in `Minicraft/Engine`.
    </Card>
    <Card title="Game Mechanics" icon="puzzle">
        High-level gameplay logic including Chunk Generation, Item Registries,
        and an Entity Component System. Located in `Minicraft/Game`.
    </Card>
    <Card title="Rendering" icon="rocket">
        An **OpenTK** render pipeline. Utilizes internal face culling, frustum
        culling, backface culling, etc. to efficiently render voxels.
    </Card>
    <Card title="Data-Driven Assets" icon="document">
        An asset system that loads Items, Blocks, and Textures dynamically from
        JSON files in the `Assets/` directory.
    </Card>
</CardGrid>
